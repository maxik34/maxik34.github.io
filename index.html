About

Welcome to my page where I try to show and tell about all the projects and things I have worked with or am working on now. I do many things alone and very often I don't finish my projects, but I still return to them from time to time.

My Projects

Many projects are still in the development stage, if you want to help with something or have suggestions, you can DM me.

The Stanley Parable: Ultimate Edition

Mod for the game The Stanley Parable (2013), which improves the game, adds new content and content from the Ultra Deluxe version of the game. Also the launcher and some improvements for modding the game.

ModDB Page

World of Goo Plus

Remaster of the game World of Goo. Basically a complete creation of all textures from scratch and improvements for modding the game.

VDCEditor

Browser extension that changes the text field for editing MediaWiki-based projects pages to a more advanced one. Originally designed for the Valve Developer Community site, but can be used in other projects.

VDC Page Download (version 0.1)

Source Renewed

SourceRenewed is a project based on the Source SDK 2013 Singleplayer. Adds some new and useful things to the engine and improves the existing ones. However, unlike Mapbase, this project is not meant to be fully backward compatible with any Source branch.

VDC Page

Windows 7+

This is a custom image of Windows 7 with various improvements, such as a dark theme (without third-party programs), improvements in the context menu, and some other improvements.

Aero+

Improved version of aero.msstyles file for Windows 7. What has been changed: improved corner rounding (more smoothed), changing old textures that do not fit into the aero style, like the language button in the taskbar or buttons in Taskband Extended UI, as well as other minor improvements.

P2CE HammerCrasher

On April 1st, the creators of Portal 2 Community Edition decided to make achievements for the Hammer and several achievements required you to crash the program a lot of times. I, for fun, decided to make a program that does this itself.

New Steelfeathers Enchanted Pack

Texture pack that i liked. This is an attempt to update it to new versions of minecraft.

TrayMusicPlayer

A small program for playing music (local files only) directly from the tray.

ActiveCollege

My Diploma program. Simplifies work with Microsoft Active Directory.

Fast_Work

A program I made in college. Our college had a problem where when you logged into your account from another computer, your desktop (files, shortcuts, etc.) on the other PC was not transferred. This program was aimed at solving this problem.

R.I.S.

My first game made on Construct Classic. The name is an acronym in Russian РИС, which means rice and stands for Работай и строй (work and build). A simple 2D game in which you need to furnish your home (room).
